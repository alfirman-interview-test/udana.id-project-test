## Technologi Used
- React js
- TailwindCSS

## Installation
If you've cloned this repository, open in terminal and typing in your terminal with command :
```
yarn install

/* Note
make sure your computer is installed Node JS and yarn.
*/
```
for installations all dependencies.

## Running
For Running :
```
yarn start
```
